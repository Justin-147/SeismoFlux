## 1. 科学边界

阶段 2 只建立三类不含异常的地震活动背景：空间均匀 Poisson、空间变化 Poisson（历史地震 KDE）和标准时空 ETAS。模型只能读取阶段 1 冻结的物理地震目录、目标无关研究区和本协议允许的时间字段；禁止读取异常特征、断层、底图、人工预测和 LocationPred 旧模型结果。

唯一例外是从 `anomaly_report_period.available_at` 读取去重后的真实报告签发日，作为后续所有模型共享的起报日历。该过程禁止读取同表其他列，且不构成异常特征。地震 Parquet、来源 Parquet、数据目录、研究区、起报清单、生产解析 fixture、外部 oracle 元数据及 `uv.lock` 的 SHA-256 均已冻结并由严格加载器交叉验证。

G1 的主目标是达到冻结目录完整度震级以上的无标记物理地震点过程。M5-6 与 M6+ 结果在阶段 7 之前仅作探索性样本水位和积分诊断；不得把重叠窗口中的同一事件重复解释为独立证据。

## 2. 时间、折与防泄漏

所有起报时刻为北京时间实际起报日的 `00:00:00`，目标窗口为 `(T,T+h]`，`h∈{7,30,90,180,365}`。开发期共有 50 个真实起报日，验证期共有 51 个；验证期最后真实起报日为 2025-06-26。每个窗口从分区第一个真实起报日起，贪心选择下一个不早于前一起报日加窗口长度的真实起报日；零事件曝光仍保留。完整日期和每个窗口的非重叠曝光均已逐项冻结在清单中。开发期真实起报日保留给后续异常模型作同日比较，阶段 2 的最终背景参数快照不得用它们评分。

四个历史目录折只用于完整度、KDE 带宽、ETAS 参数与稳定性选择。每折必须使用只拟合到该折 `fit_end_utc` 的独立参数快照，评估区间前留出至少 365 天目标清洗期。最终参数只用 2023-06-30 16:00 UTC 及以前的数据拟合；验证或折评估进行到时刻 `t` 时，拟合截止后但 `available_at<=t` 的地震可以顺序加入 ETAS 因果历史，参数不得重估。这样既允许真实在线历史更新，又不会用起报日后的地震拟合起报日前参数。

地震目录缺少权威发布时间，因此阶段 1 以发震时刻作为乐观 `available_at`。阶段 2 必须同时报告固定延迟 0、1、7 天敏感性，且不得把乐观假设写成事实。锁定测试保持空白，本阶段不得运行或查看。

## 3. 目录完整度与震级模型

完整度只使用研究区内事件，并按“历史折用该折拟合截止、验证用最终拟合截止”分别估计；300 km 缓冲事件只可作为 ETAS 父事件，禁止进入完整度。完整度采用 0.1 级分箱的最大曲率加固定 0.2 修正，再向上吸附到 `{3.0,3.2,3.5,4.0}`。主阈值是所有合格时间块和空间层估计所需的最小共同候选值；任何合格估计高于 4.0、或时间/空间均无合格层时，阶段 2 直接失败，不得扩充候选后继续看成绩。

- 时间诊断：从 1970 年固定锚点划分五年块；末尾不足五年的块只有持续至少三年且至少 200 个事件才进入主估计。
- 制度诊断：相邻块完整度差达到 0.3，或年化计数比达到 2.0 时标记为“可能的检测制度变化”，不作因果解释。
- 空间诊断：使用原点固定、目标无关的 500 km 等面积格；少于 200 个事件的格并入固定 1000 km 父格，仍稀疏则标记不可判定并使用全局主阈值，不能据震中或成绩重划区域。
- 1900–1969 的 M5+ 来源禁止混入 M3 完整度拟合。目录缺少可靠震级类型，阶段 2 仅使用保留缺失标记的统一数值震级。

未来后代震级使用闭区间截断 Gutenberg–Richter 分布，`m_min=Mc`、`m_max=9.5`，超上限观测触发硬失败；`b=log10(e)/(mean(M)-(Mc-0.05))`，`β=b ln(10)`。令 `L=m_max-Mc`，当 `|α-β|>1e-12` 时，`E[e^{α(M-Mc)}]=β/(β-α)·[1-e^{-(β-α)L}]/[1-e^{-βL}]`；否则取连续极限 `βL/[1-e^{-βL}]`。分枝率 `K` 乘该期望，必须小于 0.95。

## 4. 三类背景模型

1. **均匀 Poisson**：研究区内完整事件数除以等面积时空暴露。
2. **空间变化 Poisson/KDE**：只用研究区内地震，在等面积坐标中构建无限支持高斯混合，并把完整混合在研究区上整体归一一次，不做逐核边界修正；归一与拟合目标统一使用 12.5 km 冻结网格的裁剪单元 `representative_point×精确面积` 求积，归一后空间密度在研究区积分为 1。带宽候选固定为 75、100、150、200、300 km。每个候选在四个历史折的相同事件和补偿项上计算连续时间每物理事件信息增益，以四折等权均值最高者为最佳；对每个候选计算“候选减最佳”的四个配对折差，标准误固定为这些差的 `ddof=1` 样本标准差除以 `sqrt(4)`。候选四折均值不低于最佳均值减该配对标准误即合格，最终取通过归一和网格收敛门控的合格候选中最大带宽；数值完全相等也取最大带宽。
3. **ETAS**：背景项明确为上述冻结带宽的研究区归一 KDE，`μ` 是研究区内每日背景事件率。目标事件和补偿积分只限研究区内，并同样固定使用 12.5 km 求积；25↔12.5 km 收敛失败时该模型和该模型的 G1 资格失败。300 km 缓冲事件只作条件父历史，永不进入目标、背景 KDE 或指标。四个历史折与最终拟合都从 2000-01-01 UTC 开始。未来模拟只在研究区生成外生背景事件，落入缓冲的后代可继续触发并返回研究区，但缓冲事件不进入输出或指标。时间核为无限支持且总质量归一的 Omori–Utsu 核；3650 天只截断历史父事件且不重新归一。空间核在 300 km 径向截断后重新归一。生产实现是项目自有 NumPy/SciPy CPU 数值层，不复制外部 GPL/AGPL 代码。

ETAS 在 `log(μ),log(K),log(α),log(c),log(p-1)` 空间按索引 0–4 分别经统一 SHA-256 `optimizer_start` 命名空间派生 PCG64 初值，并在变换边界内均匀采样；不存在第二套直接种子。L-BFGS-B 固定 `tol=None,ftol=1e-12,gtol=1e-6,maxiter=500,maxfun=100000,maxls=20`，梯度对包含背景、触发和补偿的完整目标做三点数值差分，相对步长 `1e-6`；无效物理点或分枝率点返回正无穷，差分改用二阶向内单边式，仍不可用则该初值失败。至少四个初值收敛、梯度无穷范数不超过 `1e-4`、最佳三个目标函数之差除以 `max(1,|best|)` 不超过 `1e-4`，且三个变换向量逐参数极差的最大值不超过 `0.1`，才进入 Hessian 审计。Hessian 在变换尺度上做中心二阶差分，步长为 `1e-4·max(1,|z_i|)` 并对称化；任何 stencil 越界或违反分枝率都直接判 ETAS 稳定性失败，不用单边替代。最小特征值不低于 `1e-8`，条件数定义为对称正定 Hessian 的最大/最小特征值且不超过 `1e10`。任何失败都登记为 ETAS 负结果，不得事后放宽阈值；若 KDE 稳定超过均匀模型，G1 仍可通过。

## 5. 随机性、未来积分与网格

伪随机数固定为 NumPy `PCG64`，子种子由配置中的零字节分隔有序字段做 SHA-256 后取前 16 字节得到；配置同时冻结了参考摘要与整数。优化初值、未来模拟、模拟回归和 bootstrap 都只使用各自 SHA-256 命名空间；不存在独立整数子种子。四折及最终 ETAS 优化的 `model_id` 固定为 `etas/fold_1` 至 `etas/fold_4` 与 `etas/final_validation`，`issue_id` 为 `-`，复制索引为初值索引 0–4。未来模拟固定使用 `etas/final_validation`，`issue_id` 为 `validation/<冻结清单中的 YYYY-MM-DD>`，复制索引 0–127。解析模拟回归固定使用 `etas_inverse_power_cut300_v1`、`issue_id=-`、索引 0–8191。G1 bootstrap 的模型上下文只能是 `spatial_poisson_vs_uniform_poisson` 或 `etas_vs_uniform_poisson`，`issue_id` 固定为 `g1_primary_validation_2024-07-01_2025-07-01`，索引 0–1999。所有索引按八位十进制编码。单个复制最多模拟 100,000 个事件，溢出即硬失败。每个起报日、模型和复制只模拟一次 365 天完整级联，再将同一目录聚合到全部短窗口和三套网格，禁止分别重模拟。

等面积网格原点固定为 `(0,0)` 米，`column=floor(x/cell_m)`、`row=floor(y/cell_m)`，单元采用左下闭、右上开边界，并按 `row`、`column` 升序。ID 用整数毫米单元边长及带符号八位行列号，避免 12.5 的字符串差异。Shapely 与研究区求交后只保留正投影面积单元，积分点固定为裁剪几何的 `representative_point()`，单元质量为该点密度乘精确裁剪面积。25↔12.5 km 是主收敛比较，50↔25 km 是诊断比较；细格质量先聚合到对齐粗格。总量差为 `|coarse-fine|/max(|fine|,1e-12)`；密度 L1 是粗格归一质量与聚合细格归一质量的绝对差之和。阈值分别为 2% 与 5%，双零记零差异。任何网格都不得因真实震中、命中情况或高分区域局部加密。

## 6. 数值回归与外部 oracle

生产路径必须通过项目内微型解析 fixture：空历史退化为背景、单源时间核和空间核公式、背景与触发可加、强度非负、窗口积分单调、截断核归一和分枝率闭式计算。普通标量绝对/相对容差冻结为 `1e-14`/`5e-12`，对数似然绝对容差为 `1e-11`。

独立权威 oracle 是 CRAN `ETAS 0.7.2` 与 Jalilian（2019）JSS Japan 参考结果。其 tarball、R 数据、参数向量、单位、对数似然与 AIC 均记录在独立元数据 fixture 中；外部执行为可选隔离核验，不进入 Python 生产依赖，也不复制 GPL 代码。自有模拟解析回归使用 8,192 个复制检查后代均值、方差、零后代率和直接后代 PIT；隔离的 `lmizrahi/etas 3.2` 比较各 4,096 个复制的后代数、均值/方差、零后代率、出现第二代概率，以及首个直接后代的时间、半径和震级两样本 KS。精确阈值在 YAML 中冻结。外部 oracle 不可用时只能登记“未运行”，不能豁免生产解析回归、性质测试和自有模拟回归。

## 7. G1 与阶段出口

G1 只有一个主终点：验证日历段 `(2024-07-01 00:00, 2025-07-01 00:00] Asia/Shanghai` 上，区内 `M>=Mc` 无标记事件的连续时间顺序条件点过程对数似然。信息增益同时包含事件对数条件强度和研究区积分补偿项，以区内独立物理目标事件数为分母；无目标事件时只报告证据不足。五个预测窗口不聚合、不参与 G1，只按真实起报日的非重叠曝光逐窗报告；零事件曝光保留补偿项，某窗总目标为零时该窗指标未定义。G1 不能拼接不同模型的局部通过：同一个空间变化 Poisson 或 ETAS 必须同时满足验证信息增益为正、四个历史开发折中至少三个为正和该模型数值稳定，G1 才通过。最终模型选择的候选池只能包含拟合成功、通过该模型全部数值稳定性门控且通过网格收敛的模型；失败模型不得参与验证最佳值、一标准误阈值或最终选择。因此 ETAS Hessian 审计失败时不会阻碍合格 KDE 独立通过 G1。候选池内先按验证期每物理事件信息增益找最佳；简单度固定为均匀 Poisson、空间 Poisson、ETAS。候选与验证最佳模型在每折相同事件和补偿上做配对差，标准误为四个配对差的 `ddof=1` 样本标准差除以 `sqrt(4)`；候选验证 IG 不低于最佳验证 IG 减该标准误即进入一标准误集合，最后取简单度顺序中的首个模型。

阶段 2 必须交付背景模型注册表、同切分回测指标、代表起报日的条件强度图、背景/触发分解、参数不确定性、完整度敏感性、模拟与数值回归结果。输出 ID 使用 `seismoflux_canonical_json_v1`：Unicode NFC、键按码点排序、有限浮点用带类型的 `float.hex()` 对象、项目路径转 POSIX 相对形式，并把协议、输入哈希、模型参数、代码提交和 `uv.lock` 哈希共同寻址。内部浮点标签键 `$seismoflux_type` 为保留命名空间，调用方映射不得自行提供，以保证浮点与普通映射不会产生同一规范字节。输入哈希键必须且只能是环境锁、数据目录、地震数据、研究区、起报清单、生产 fixture 和 oracle 元数据七项，`uv_lock_sha256` 必须与环境锁项相同；缺项、多项或非小写 SHA-256 均拒绝。产物先写同父目录临时区，逐文件记录相对路径、字节数、SHA-256 和媒体类型，刷盘后原子改名；同一 ID 的字节不一致即硬失败，完全一致也只核验不重写。全部测试与独立复核通过后，才提交、推送、通过远端 CI 并打 `v0.2.0-background-baselines`。若 G1 失败，则提交可信负结果并停止，不进入阶段 3。

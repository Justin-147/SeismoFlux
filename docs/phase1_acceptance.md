## 冻结决策

### 研究区

研究区来自 `PlotBD/CN-border-L1.gmt`，按“有效闭合环中等面积最大；相同面积时源段号最小”的确定性规则选择第 1 段。它表示中国连续大陆，排除海南、台湾及其他离岛；边界使用 `covers` 判定并计入区内。选区过程不读取目标事件或真实震中。

### 时间可用性

- 异常记录的 `available_at` 为文件名报告日与行内报告日较晚者的次日 `00:00:00 UTC+08:00`。
- 地震目录没有权威时区，原始时间按固定 `UTC+08:00` 假设转换为 UTC，不采用历史 LMT 或夏令时变化。
- 地震目录没有发布时间，逐条设置 `available_at = origin_time_utc` 作为乐观假设并保留质量标记。
- 异常起止时间源字段只有日期，统一编码为固定 `UTC+08:00` 当日 00:00 的带时区时间并逐条标记，不把它解释为精确时刻。
- 任何起报时刻只允许使用 `available_at <= T` 或 `origin_time <= T` 的记录；未来异常结束时间、未来报告状态和未来地震不得回填。

### 地震去重

- 跨目录自动合并：时间差不超过 5 秒、测地距离不超过 5 km、震级差不超过 0.1，且候选图两侧节点度都为 1。
- 人工复核候选：时间差不超过 300 秒、距离不超过 50 km、震级差不超过 0.5。
- 共输出 3,006 个复核候选；2,882 个满足无歧义自动合并条件，另 124 个仅供人工复核，不静默合并。
- 同源目录只折叠精确语义重复；禁止模糊同源合并、歧义合并和传递合并。

### 地质与底图快照

当前断层、长期危险性、真实迹线和底图快照只能证明在 `2026-07-13 00:00:00 UTC+08:00` 可用，历史可用时间无法证明。所有相关记录均标记为历史模型不可用；在取得带时间的历史版本前，不得将当前快照用于更早起报日的历史回测。

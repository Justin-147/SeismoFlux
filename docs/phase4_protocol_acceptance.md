## 数据与模型边界

- 背景基准固定为已验收的 75 km Gaussian KDE 空间 Poisson 模型；ETAS 数值稳定性未达到可评估状态，必须如实显示为 `not_evaluable`，不能伪填成绩。
- 第一模型固定为简单、可解释的 ridge Poisson 点过程；不允许在简单模型通过门控前进入树模型或神经网络。
- 覆盖控制、单期异常和动态轨迹三类增量共享冻结的预处理、正则化和拟合规则；相对强度、评分和顺位不得称为绝对发震概率。
- 每个折只在共同的 7 日拟合暴露上拟合一次预处理器和一组异常系数，再积分到 7、30、90、180、365 日窗口。
- 背景与各震级目标率头只从背景拟合暴露估计一次并冻结；每次时间或空间置乱只重拟合预处理器与异常系数，背景和目标率头原样复用。
- M5--6 是主要证据层；M6+ 若拟合期为零事件则保持零强度并从相应推断中排除，不加伪计数。
- 7、30、90 日参与主要 G2/G3 门控；180、365 日只报告证据不足，不能挽救门控。
- 任一起报暴露严格使用 `(T,T+h]`，不得使用 `T` 之后的异常报告、地震或人工结果。
- 真实震中不得生成候选、加密网格或区域边界；事件只在冻结 25 km 网格上进行确定性映射并用于评分。
- 局部高 Mc 只影响相应背景支持解释，不得改变其它区域的异常候选、异常特征或空间置乱边界。
- 人工预测地点、时间、震级和结果继续禁止作为特征或标签；断层增量推迟到阶段 5。

## 冻结评估与对照

- 主比较：冻结背景、背景加覆盖控制、背景加单期异常、背景加动态异常。
- 零假设对照：时间置乱 1,000 次、空间实体坐标对置乱 1,000 次；置乱映射由类型化随机上下文和冻结输入封印唯一确定。
- 置信区间：按物理事件、震级层和五窗口成员关系分层的 2,000 次 bootstrap；同一重采样权重跨模型、指标和窗口复用。
- 报警面积：同面积诊断固定 600,000 km²；同召回比较使用冻结的 0 至 960,000 km²、步长 625 km² 预算格，不做插值。
- G2 要求动态异常相对背景在 M5--6 的 7/30/90 日信息增益、置乱显著性及受控报警面积改善共同通过；G3 要求动态轨迹相对单期异常在联合开发折上稳定为正。
- 任一门控失败都形成可信负结果；不得在读取正式验证目标后继续调参，也不得用 180/365 日或单一区域结果补救失败。

## 空间置乱公开边界

原始 L1/L2 线文件许可状态为 `unknown_no_redistribution`，因此未进入 Git。坐标、逐格映射、实体级映射、连接线和派生几何只保留在被忽略的 `data/interim/stage4/` 中。公开空间清单只包含算法、哈希和聚合计数，不包含经纬度、实体 ID、逐格记录、WKT/WKB 或 GeoJSON。

冻结拓扑把 33 条输入线分割、裁剪并补连接为 65 个构造分层；25 km 查询网格共有 15,697 个唯一格。65 个分层中 39 个包含查询格、26 个为零格分层，零格和单格分层均有明确失败关闭语义。165,984 个实体状态中 165,841 个可分层、143 个不合格；1,180 个状态位于边界外并按冻结最近分层规则处理。所有这些数字来自目标无关的异常状态与构造线，不读取地震震中。

## GPU 与资源规则

- CPU `float64` 是科学权威实现。
- GPU 只作为可选加速后端，不能改变算法、精度、随机映射、收敛判据或结果身份。
- GPU 必须先通过与 CPU 的严格数值等价、身份安慰剂、重复运行和不同工作线程一致性测试；未通过时自动回退 CPU，不能阻断或放宽科学门控。
- 大型任务最多使用 12 个工作进程，至少保留 2 个物理核心，禁止嵌套并行。
